# 路由设计

```bash
| 路径          | 场景 / 页面说明      | 对应组件                         |
| ----------- | -------------- | ---------------------------- |
| `/`         | 首页（品牌入口、CTA）   | `app/page.tsx`               |
| `/writing`  | AI 写作场景        | `<Editor mode="writing" />`  |
| `/research` | AI 研究场景        | `<Editor mode="research" />` |
| `/learning` | AI 学习场景        | `<Editor mode="learning" />` |
| `/planning` | AI 规划场景        | `<Editor mode="planning" />` |
| `/creative` | AI 创意创作        | `<Editor mode="creative" />` |
| `/explore`  | AI 模板 / 命令面板入口 | `ExplorePage`                |
| `/search`   | 图谱搜索 / 语义搜索    | `SearchPage`                 |
```
